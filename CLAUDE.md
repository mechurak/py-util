## Project Overview

A Python utility project (py-util) — 개인용 유틸리티 스크립트 모음. uv를 패키지 매니저로 사용하며 Python 3.12+.

### Modules

- `src/srt2lrc/` — TurboScribe SRT 자막 파일을 안드로이드 플레이어용 LRC 가사 파일로 변환
- `src/toc_extractor/` — 교보문고 등 도서/강의 페이지에서 목차를 추출하여 Logseq 페이지로 변환
